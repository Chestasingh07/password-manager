# Secure CLI Password Manager

## Description
This is a command-line based password manager that allows users to store, view, and search passwords securely.

## Features
- Save passwords
- View stored passwords
- Search passwords

## Requirements
- Python 3

## How to Run
1. Open terminal
2. Navigate to project folder
3. Run:
   python main.py

## Files
- main.py : main program
- passwords.txt : stores passwords
